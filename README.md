# qpm.vn
Some variants of QPM tailored for Vietnam.

1. Baseline model of 4 main equations SOE-NK
2. Ext1: Disintegrated the model into core and non-core CPI
3. Ext2: Adding credit as a policy instruments
4. Ext3: Adding central banks' credibility
5. Ext4: Adding bank's lending tightening
6. Ext5: Adding fiscal policies
7. Ext6: Adding labor market indicators
8. Ext7: Weather shocks on Agriculture and Food prices

For `IRIS_Toolbox` and `utils` folders, download them from here: [https://github.com/thanhqtran/imf.qpm](https://github.com/thanhqtran/imf.qpm).
